📐 Derivative Formulas

Calculus I - Learn & Practice

📘 Basic Rules

Power Rule
d/dx[xⁿ] = n·xⁿ⁻¹
Example: d/dx(x³) = 3x²
Constant
d/dx(C) = 0
Example: d/dx(5) = 0
Linear
d/dx(kx) = k
Example: d/dx(3x) = 3

🔺 Trigonometric

Sine
d/dx(sin x) = cos x
Example: d/dx(sin 2x) = 2cos 2x
Cosine
d/dx(cos x) = -sin x
Example: d/dx(cos 3x) = -3sin 3x
Tangent
d/dx(tan x) = sec² x
Example: d/dx(tan x²) = 2x sec²(x²)
Cotangent
d/dx(cot x) = -csc² x
Example: d/dx(cot 2x) = -2csc²(2x)
Secant
d/dx(sec x) = sec x · tan x
Example: d/dx(sec 3x) = 3sec(3x)tan(3x)
Cosecant
d/dx(csc x) = -csc x · cot x
Example: d/dx(csc x²) = -2x csc(x²)cot(x²)

📈 Exponential & Logarithmic

eˣ
d/dx(eˣ) = eˣ
Example: d/dx(e²ˣ) = 2e²ˣ
aˣ
d/dx(aˣ) = aˣ · ln a
Example: d/dx(2ˣ) = 2ˣ · ln 2
ln x
d/dx(ln x) = 1/x
Example: d/dx(ln 2x) = 1/x

📋 Rules

Sum Rule
d/dx(f ± g) = f' ± g'
Example: d/dx(x² + sin x) = 2x + cos x
Product Rule
d/dx(f·g) = f'·g + f·g'
Example: d/dx(x·sin x) = sin x + x·cos x
Quotient Rule
d/dx(f/g) = (f'·g - f·g')/g²
Example: d/dx(sin x / x) = (x·cos x - sin x)/x²
Chain Rule
d/dx[f(g(x))] = f'(g)·g'(x)
Example: d/dx[sin(x²)] = cos(x²)·2x
